# Prison OP Server Configuration

Este proyecto contiene configuraciones avanzadas para servidores de Minecraft en la modalidad **Prison OP**.

## Descripción

Prison OP es una modalidad centrada en la progresión mediante minería y comercio. Los jugadores comienzan con herramientas básicas y deben trabajar para avanzar a través de rangos y minas cada vez más poderosas.

El sistema está diseñado para mantener una progresión constante y motivar a los jugadores a mejorar su equipamiento.

## Sistema de minas

Las minas están diseñadas para ofrecer distintos tipos de recursos dependiendo del nivel del jugador.

Tipos de minas:

- mina básica
- mina intermedia
- mina avanzada
- mina legendaria

Cada mina incluye bloques especiales que pueden generar recompensas raras.

## Sistema de economía

El núcleo del servidor gira en torno a la economía basada en minería.

Los jugadores pueden:

- vender minerales
- comerciar con otros jugadores
- invertir en mejoras
- participar en eventos económicos

## Sistema de encantamientos

Las herramientas pueden ser mejoradas mediante encantamientos personalizados que aumentan la eficiencia de minería.

Ejemplos:

- explosión minera
- duplicación de bloques
- multiplicador de drops
- velocidad de minería

## Eventos especiales

Para mantener la actividad del servidor se incluyen eventos periódicos como:

- minas temporales
- multiplicadores de venta
- invasiones de mobs
- jefes de minería

## Plugins utilizados

Este sistema utiliza plugins populares de la comunidad:

- LuckPerms
- PlaceholderAPI
- Vault
- AdvancedEnchantments

## Objetivo

El objetivo de este proyecto es mostrar una estructura profesional de configuración para servidores Prison OP, con sistemas de progresión claros y balanceados.
